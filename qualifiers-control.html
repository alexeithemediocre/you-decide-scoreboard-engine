Danger zone

Open qualifiers.html in another window (that's the one you capture in OBS / share on stream). Both pages must be open in the same browser on this computer. Using Firefox or Safari? Don't open the files directly — double-click Start scoreboard.command (Mac) or Start scoreboard.bat (Windows) instead: those browsers only sync the pages when served from http://localhost. (The scripts need Python 3 — most Macs already have it; on Windows install it from python.org.)
